Exemples de modèles

  • GPT-4 Turbo
  • Claude Sonnet 4.5
  • Gemini 2.0 Flash

Cas d'usage

  • Analyse de documents
  • Génération de code
  • Tâches multi-outils
  • Planification simple

Coût

Raisonnement étendu

(10-120 secondes)

Fonctionnement

Exploration multi-branches avec validation interne et backtracking.

Exemples de modèles

  • GPT-o1
  • GPT-o3
  • Claude Sonnet 4.5 (mode étendu)

Cas d'usage

  • Recherche scientifique
  • Problèmes mathématiques
  • Stratégie complexe
  • Raisonnement multi-étapes

Coût

15-60$ / 1M tokens
